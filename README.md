# Addressable Demo
本项目用于练习Unity中的Addressable功能。

## 项目目标
制作出三个场景，使用Addressable分别打包成三个AssetsBundle，实现按需下载更新包功能

## 项目需求
- 制作场景（已完成，0.5h）
- 制作UI界面（进行中，1h）
- 实现展示包体大小、下载进度、包体加载与卸载等功能
